THIET KE API DE XUAT

Du an: Ban hoc thong minh
Huong API: REST API
Dinh dang du lieu: JSON
Xac thuc: Access token + Refresh token
Phan quyen: teacher | parent | student
Tien to API de xuat: /api/v1

I. NGUYEN TAC API
- Bai hoc la don vi duoc giao cho hoc sinh.
- Moi bai hoc gom nhieu hoat dong.
- Frontend/PWA khong goi Gemini truc tiep bang API key that; backend la noi goi Gemini.
- Moi API quan trong nen tra ve request_id de de doi chieu log server.
- Tat ca loi 5xx can duoc ghi log o backend.

II. CHUAN RESPONSE DE XUAT
1. Thanh cong
{
  "success": true,
  "data": {},
  "message": "OK",
  "request_id": "req_xxx"
}

2. That bai
{
  "success": false,
  "message": "Noi dung loi",
  "error_code": "SOME_ERROR",
  "request_id": "req_xxx"
}

III. AUTH API
1. POST /api/v1/auth/login
- Dang nhap bang email/phone + password.
- Tra ve access_token, refresh_token, user info.

2. POST /api/v1/auth/refresh
- Lam moi access token.

3. POST /api/v1/auth/logout
- Dang xuat.
- Vo hieu hoa refresh token neu can.

4. POST /api/v1/auth/change-password
- Doi mat khau.

5. GET /api/v1/auth/me
- Lay thong tin user dang dang nhap.
- Tra ve role va profile co lien quan.

IV. USER / PROFILE API
1. GET /api/v1/users/me/profile
- Lay profile hien tai theo role.

2. PUT /api/v1/users/me/profile
- Cap nhat profile cua user hien tai.

3. GET /api/v1/students/:studentId
- Lay chi tiet hoc sinh.
- Teacher xem hoc sinh cua minh.
- Parent chi xem hoc sinh da duoc lien ket.

4. POST /api/v1/students
- Teacher tao hoc sinh moi.

5. PUT /api/v1/students/:studentId
- Teacher cap nhat ho so hoc sinh.

6. GET /api/v1/students
- Teacher xem danh sach hoc sinh cua minh.
- Ho tro filter theo muc do, lop, tu khoa.

V. CLASS API
1. GET /api/v1/classes
- Teacher lay danh sach lop cua minh.
- Co filter active/archived.

2. POST /api/v1/classes
- Teacher tao lop moi.

3. GET /api/v1/classes/:classId
- Lay chi tiet lop.

4. PUT /api/v1/classes/:classId
- Sua thong tin lop.

5. DELETE /api/v1/classes/:classId
- Xoa hoac archive lop.

6. GET /api/v1/classes/:classId/students
- Lay danh sach hoc sinh trong lop.

7. POST /api/v1/classes/:classId/students
- Them hoc sinh vao lop.
- Body: student_ids[] hoac 1 student_id.

8. DELETE /api/v1/classes/:classId/students/:studentId
- Xoa hoc sinh khoi lop.

VI. SUBJECT API
1. GET /api/v1/subjects

2. POST /api/v1/subjects

3. PUT /api/v1/subjects/:subjectId
- Sua mon hoc.

4. GET /api/v1/classes/:classId/subjects
- Lay mon hoc dang cau hinh trong lop.

5. POST /api/v1/classes/:classId/subjects
- Gan mon hoc vao lop.

6. DELETE /api/v1/classes/:classId/subjects/:subjectId
- Bo mon hoc khoi lop.

VII. TOPIC API

VIII. LESSON API
1. GET /api/v1/lessons
- Teacher lay danh sach bai hoc.
- Filter theo mon hoc, muc do, chu de, trang thai.

2. POST /api/v1/lessons
- Tao bai hoc moi.
- Body gom: title, subject_id, description, primary_level, topic_ids, estimated_minutes...

3. GET /api/v1/lessons/:lessonId
- Lay chi tiet bai hoc va danh sach hoat dong ben trong.

4. PUT /api/v1/lessons/:lessonId
- Sua bai hoc.

5. DELETE /api/v1/lessons/:lessonId
- Xoa/luu tru bai hoc.

6. POST /api/v1/lessons/:lessonId/publish
- Chuyen trang thai publish.

7. POST /api/v1/lessons/:lessonId/archive
- Luu tru bai hoc.

IX. LESSON ACTIVITY API

3. GET /api/v1/activities/:activityId

4. PUT /api/v1/activities/:activityId
- Sua hoat dong.

5. DELETE /api/v1/activities/:activityId

6. POST /api/v1/lessons/:lessonId/activities/reorder

X. MEDIA API

3. DELETE /api/v1/media/:assetId

5. DELETE /api/v1/activities/:activityId/assets/:assetId

XI. ASSIGNMENT API
1. GET /api/v1/assignments
- Teacher lay danh sach bai hoc da giao.
- Filter theo lop, mon hoc, student, status.

2. POST /api/v1/assignments
- Giao bai hoc.
- Body: lesson_id, class_id, subject_id, target_type, student_ids[], due_at, required_completion_percent.

3. GET /api/v1/assignments/:assignmentId
- Xem chi tiet mot lan giao bai.

4. PUT /api/v1/assignments/:assignmentId
- Sua han/doi tuong giao bai neu chua hoc.

5. POST /api/v1/assignments/:assignmentId/close
- Dong assignment.

XII. STUDENT LEARNING API
1. GET /api/v1/my/classes
- Hoc sinh lay danh sach lop cua minh.

2. GET /api/v1/my/subjects

3. GET /api/v1/my/assignments

4. GET /api/v1/my/assignments/:assignmentId

- Tra ve cau truc bai hoc + danh sach hoat dong.

5. POST /api/v1/my/assignments/:assignmentId/start
- Bat dau bai hoc.

6. POST /api/v1/my/activities/:activityId/submit
- Nop ket qua 1 hoat dong.
- Body co the gom: response_mode, selected_answer, drag_result, transcript, confidence, duration_seconds, raw_result_json.

7. POST /api/v1/my/assignments/:assignmentId/complete
- Danh dau hoan thanh bai hoc.

8. GET /api/v1/my/progress
- Hoc sinh xem tien do cua minh.
