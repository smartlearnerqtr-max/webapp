# Backend Flask

Backend API cho du an Ban hoc thong minh.

## Chay local

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
flask --app run.py run
```

## Endpoint co san

- `GET /api/v1/health`
- `POST /api/v1/auth/login` (placeholder)
