DE XUAT STACK CONG NGHE

Du an: Ban hoc thong minh
Muc tieu: webapp/PWA dung tot tren desktop, cai dat duoc tren Android, backend Python Flask, mo rong AI Gemini, logging va offline/sync.

I. DE XUAT STACK TONG THE
1. Frontend
- React + TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- vite-plugin-pwa
- CSS thu cong hoac Tailwind o giai doan sau neu can

2. Backend
- Python 3.12+
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Flask-JWT-Extended
- Flask-CORS

3. Database
- PostgreSQL

4. File storage
- S3-compatible storage
- MinIO local dev, S3/R2 production

5. AI
- Gemini API
- Model mac dinh: gemini-2.5-flash
- Backend Flask la noi duy nhat goi Gemini bang API key that

6. Logging va monitor
- Python logging hoac structlog/Pino-style JSON logging neu can
- Bang server_logs trong database
- Sentry hoac OpenTelemetry o production neu can

7. Test
- Frontend: Vitest + Testing Library
- Backend: pytest
- E2E: Playwright

II. VI SAO CHON STACK NAY
1. React + Vite + PWA
- Nhanh, gon, hop voi giao dien cho desktop va Android.
- De chia dashboard giao vien/phu huynh va giao dien hoc sinh.

2. Flask
- Hop voi backend Python gon nhe, linh hoat, de mo rong theo module.
- Rat hop khi can tich hop AI, logging, upload, readiness va rule nghiep vu.

3. PostgreSQL
- Tot cho quan he nhieu-nhieu va bao cao tien do, readiness, parent-student, class-subject.

V. BAO MAT
- JWT access + refresh token
- Phan quyen theo role teacher, parent, student
- Encrypt Gemini API key truoc khi luu
- Khong log password, token, API key plaintext
- Rate limit cho auth, AI, upload neu can

VI. PWA VA OFFLINE
- manifest.webmanifest
- service worker tu vite-plugin-pwa
- cache app shell
- cache danh sach bai hoc va tai nguyen da tai
- AI va test key van can internet

VII. AI GEMINI
- Frontend gui request den Flask backend
- Backend doc Gemini API key da ma hoa cua user
- Backend goi gemini-2.5-flash
- Backend ghi log request_id, model_name, error_code, timeout khi co loi

VIII. LOGGING
- Log backend can co request_id
- Ghi vao stdout/file va bang server_logs
- Co dashboard log ky thuat de truy vet loi

XI. STACK CHO MVP TOT NHAT
- Frontend: React + Vite + TypeScript + React Router + TanStack Query + Zustand + vite-plugin-pwa
- Backend: Flask + Flask-SQLAlchemy + Flask-Migrate + Flask-JWT-Extended
- Database: PostgreSQL
- AI: Gemini qua backend Flask
- Log: Python logging + server_logs
- Test: Vitest + pytest + Playwright
